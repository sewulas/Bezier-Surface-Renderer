# Bezier Surface Renderer

Interactive renderer for **Bezier surfaces** implemented as part of a **Computer Graphics course project**.

The program visualizes parametric surfaces defined by control points and allows the user to manipulate the surface structure interactively. The project focuses on implementing fundamental concepts of **geometric modeling, parametric surfaces, and rendering algorithms**.

---

# Features

### Surface visualization

* Rendering of **Bezier surfaces**
* Visualization of the **control mesh**
* Smooth surface approximation using tessellation

### Surface representation

* Parametric representation of the surface
* Surface defined by a grid of control points
* Surface evaluation using Bernstein polynomial basis
